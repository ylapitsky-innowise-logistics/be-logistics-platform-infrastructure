## 🛠 Быстрый старт

Инфраструктура платформы поднимается локально через Docker Compose. 
Конфигурационный файл расположен в подпапке `local/`.

Для фонового запуска всех баз данных и служебных систем выполните из корня репозитория:

```Bash
docker-compose -f local/docker-compose.yml up -d
```

> **Важно:** Подробные инструкции по траблшутингу, мягкой остановке 
> и полному сбросу инфраструктуры находятся в файле [Runbook](docs/RUNBOOK.md).
