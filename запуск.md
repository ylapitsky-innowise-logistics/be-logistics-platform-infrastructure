## Запуск инфраструктуры

```bash
# Запустить все сервисы инфраструктуры
docker-compose up -d

# Проверить статус
docker-compose ps

# Посмотреть логи конкретного сервиса
docker-compose logs -f postgres

# Остановить инфраструктуру
docker-compose down

# Остановить с очисткой данных
docker-compose down -v
```
